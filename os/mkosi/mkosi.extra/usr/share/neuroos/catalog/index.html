NeuroOS Catalog (MVP)

This OS scales “app count” by combining Web/PWA + Flatpak (+ Android later). This page is offline and ships in the image.

Launch apps

Use the launcher in the desktop session, or open a terminal. 
wofi --show drun
Suggested: bind a key to launcher (see /etc/sway/config.d in the image).

Install Flatpak apps (Linux ecosystem)

Flathub is pre-configured. Example installs: 
flatpak install -y flathub org.mozilla.firefox
flatpak install -y flathub com.valvesoftware.Steam
flatpak install -y flathub org.gimp.GIMP
Helper wrapper (same actions, easier to remember): 
neuroos-flatpak install org.mozilla.firefox
neuroos-flatpak update
List installed apps: 
flatpak list --app

Install PWAs (Web ecosystem)

Open a site in Chromium and use the “Install app” action (varies by site). Examples: 
https://music.youtube.com
https://calendar.google.com
https://app.slack.com
Goal: treat each installed PWA as an “app” with its own storage and permissions.

Android apps (later)

Planned: Waydroid-style Android container, presenting apps as Wayland windows.
See the repo docs for the security posture and integration plan.
